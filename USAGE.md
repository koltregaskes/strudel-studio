# Usage

1. Start in `Simple View` when you want a one-screen console, or switch to `Advanced View` when you want tabbed workstation editing.
2. Press `Play` to start the local engine.
3. If the studio seems silent, use `Audio Check` in `Audio Diagnostics` before assuming playback is broken.
4. Set the session title, tempo, key, and notes in `Project Shelf` so the workspace reflects the track you are building.
5. Shape the song in `Track Structure` by editing section names, scenes, and bar counts, then drag timeline blocks to reorder them or drag the right edge to resize them.
6. Use the `Radio Edit`, `DJ Tool`, and `Extended Mix` templates when you want to swap structure quickly without rebuilding from scratch.
7. Use `Scene Arranger` to jump to a musical mood quickly.
8. Use `Beat Designer` for kick, snare, and hat patterns, then use `Sequencer Utilities` to shift, densify, thin, or clear the active lane without rebuilding the groove from scratch.
9. Capture strong ideas in `Pattern Rack` so one project can hold multiple patterns and scene variations.
10. Mix in the `Mixer` with level, mute, solo, and pan controls.
11. Work across four sample lanes:
    vocal chops, FX hits, texture beds, and perc one-shots.
12. Use `Sample + Clip Library` to review what is loaded in each lane, audition it, clear it, or aim the browser at the right target lane.
13. Use `Sample Browser` to apply template lane ideas without shipping copyrighted audio.
14. Open `Workspace` in the lower deck or `Code + AI` in `Advanced View` when you want direct Strudel editing or local AI help.
15. Use the local AI panel for composition help, explanation, and code generation from a localhost model endpoint, and use `Detect Local AI` or `Test Connection` before your first prompt.
16. Use `Export Dock` when you want a live JSON preview of the current session, a quick copy-to-clipboard export, or a download-ready project file.
17. Press `Update` after code edits when you want to evaluate the current Strudel sketch.
18. Save named local snapshots in `Project Shelf`, or export/import full project JSON.
19. Use `Record` to capture the local mix and export a take.
