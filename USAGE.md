# Usage

1. Click `Play` to start local Web Audio playback.
2. Build the song structure in `Track Structure` by adding sections, changing their scene, and setting how many bars each one lasts.
3. Adjust track levels, mute states, tempo, and effect sliders to reshape the live preview.
4. Use `Beat Designer` to click in kick, snare, and hat steps, or load a rhythm preset and then tweak it.
5. Import a vocal sample, place chops on the 16-step grid, and tune the slice start, end, and playback rate.
6. Import an FX or stab sample to build a second rhythmic one-shot lane.
7. Open `Show Code` to view or edit the generated Strudel sketch.
8. Use `Example Library` to load ready-made coded starting points into the editor.
9. Use `Update` to refresh the exported Strudel code and attempt a Strudel-side evaluation when the runtime is available.
10. Use `Export`, `Save Project`, and `Export Project` to keep your work.
