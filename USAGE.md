# Usage

1. Click `Play` to start local Web Audio playback.
2. Build the song structure in `Track Structure` by adding sections, changing their scene, and setting how many bars each one lasts.
3. Adjust track levels, mute states, tempo, and effect sliders to reshape the live preview.
4. Import a vocal sample, place chops on the 16-step grid, and tune the slice start, end, and playback rate.
5. Open `Show Code` to view or edit the generated Strudel sketch.
6. Use `Example Library` to load ready-made coded starting points into the editor.
7. Use `Update` to refresh the exported Strudel code and attempt a Strudel-side evaluation when the runtime is available.
8. Use `Export`, `Save Project`, and `Export Project` to keep your work.
