# Setup

1. Use `start.cmd` for the easiest local launch, or open `index.html` in a modern browser.
2. The page loads `@strudel/web` from `unpkg.com` for code generation support, but playback comes from the built-in Web Audio engine.
3. Allow audio playback in the browser when prompted.
4. If the external Strudel script cannot load, local playback still works and you can continue using the control surface.
5. The arrangement editor, example library, project save/import/export, vocal sampler, and FX sampler all run locally in the browser.
